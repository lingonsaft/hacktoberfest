Contribution Rules

  • Project must work when opening the index.html file
  • You are allowed to make pull requests that break the rules. We just won't merge them ;).
  • Do NOT add any build steps e.g npm install (we want to keep this a simple static site).
  • Please Do NOT remove Videos, Rules, FAQ or any other helpful content.
  • Styling / code can be pretty, ugly or stupid, big or small as long as it works.
  • Add your name to the contributors.html file.

Frequently Asked Questions

Who can contribute?

Anyone with a Github account and who signed up for Hacktoberfest 😄

Are you getting paid for this?

Sadly no. However, we think we should. This is 100% unofficial, and we do it for fun, fame, and glory.

Who are you, and why are you doing this?

We are two programmers from Sweden 🇸🇪 Richard and Benny. We are doing this because we love Open source and Hacktoberfest. We want to make it easier for people to get started with Hacktoberfest and Open source.

Why are you not using DigitalOcean?

Because we only know JavaScript and suck at servers. We use now instead.

Are you not the guys from that failed CodeCardCodingCards kickstarter?

Yes...
