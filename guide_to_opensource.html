How to Contribute to a Open Source project for HacktoberFest 2018


This is a small guide for how to contribute to an open source project and win HacktoberFest 2018



  • Click the Fork button on the upper right-hand corner. This will save the repo to your GitHub account

  • You now have a copy of the repo you just forked, available on your GitHub account

  • Clone the repo locally by running git clone git@github.com: your github username>/guides-github-pull-request.git in your terminal
  • Open this repo in your terminal and run the following command to create a new branch: git checkout -b add-your-name
  • You're now working on a new branch, on a repo that you can commit to , go ahead and make some changes and the commit those Changes git push origin add-your-name

  • Go to the shared repo on GitHub and click on the green compare & pull request button
  • Click on 'create pull request'

  • Congrats! the pull request has been made , now it can be seen by the author and can be accepted by him.
